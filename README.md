# Chirkut Gift Shop

A modern gift shop e-commerce demo built with React + Vite + TypeScript + Tailwind CSS + Express.

## Features

- Product listing with categories
- Shopping cart & wishlist
- Product detail pages
- User profile
- Responsive design (mobile-first)
- Admin panel placeholder
- Vercel-ready API

## Getting Started

```bash
npm install
npm run dev
```

This starts both the Vite frontend and the Express API server.

## Scripts

- `npm run dev` - Start development servers
- `npm run build` - Build for production
- `npm run preview` - Preview production build

## Tech Stack

- React 19 + TypeScript
- Vite 7
- Tailwind CSS 4
- Framer Motion
- Lucide Icons
- Express + CORS
- Sonner (toasts)

Demo app for educational purposes.
